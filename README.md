# ATC Career Guide

A comprehensive guide to becoming an Air Traffic Controller, covering eligibility, the hiring process, compensation, and preparation tips.

## Features

- Detailed information about ATC career requirements
- Step-by-step hiring process breakdown
- Salary and benefits overview
- ATSA test preparation guide
- Valuable resources and study tips

## Viewing the Guide

You can view the guide by opening `index.html` in your web browser or by visiting the GitHub Pages site once deployed.

## Deployment

This guide is designed to be hosted on GitHub Pages. To deploy:

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select the `main` branch and `/ (root)` folder
4. Click "Save"
